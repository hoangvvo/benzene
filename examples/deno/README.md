# deno example

## How to use

Download the example

```bash
curl https://codeload.github.com/hoangvvo/benzene/tar.gz/main | tar -xz --strip=2 benzene-main/examples/deno
cd deno
```

Run it:

```bash
deno run --unstable --allow-net server.ts
```
